# 手相与面相辅助分析规则

用于用户上传手掌图或正面头像时，与八字结论交叉验证。

## 输入要求

- 八字信息仍为必填。
- 手相图建议左右手各一张，掌纹清晰。
- 面相图需正面、自然光、无遮挡。
- 图像不清晰时只做有限度观察，并提示重拍。

## 手相观察点

| 特征 | 观察方向 | 命理映射 |
|---|---|---|
| 生命线 | 长短、深浅、断续 | 生活节律与精力基础 |
| 智慧线 | 清晰度、分叉 | 思维方式与学习路径 |
| 感情线 | 曲直、深浅、横切 | 情绪表达与关系波动 |
| 事业线 | 起点、连续度 | 职业稳定性与阶段变化 |
| 掌丘 | 饱满度 | 五行倾向补充 |

## 面相观察点

- 三停：额部、中庭、下庭，分别看早中晚阶段倾向。
- 五官：眉眼鼻口耳，对应表达、财帛、食禄、关系、祖荫等传统宫位。
- 气色：只描述明暗与状态，不做医学判断。

## 八字交叉验证

输出必须区分：
- 印证：图像特征与八字结论一致。
- 补充：图像提供八字未展开的信息。
- 矛盾：图像与八字显示不同倾向，需要保守解释。

## 合规边界

- 禁止判断疾病、寿命、容貌优劣。
- 不分析未成年人照片。
- 不做人格贬低或歧视性评价。
