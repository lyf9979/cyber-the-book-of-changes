## 三、十大维度分析

### 1. 性格特质

基于日主 + 日支 + 月令 + 透干十神的具体分析。

至少 150 字，要指出命中具体字的作用。

### 2. 感情运势

基于配偶星（男财女官）与配偶宫（日支）的分析。

包含：
- 感情模式（早婚/晚婚/稳定/波动）
- 潜在问题
- 婚姻时机倾向

### 3. 家庭与六亲

分别分析：
- 与父母的缘分（年月柱 + 偏财/正印）
- 与兄弟姐妹（比劫）
- 与子女（男看官杀、女看食伤 + 时柱）

### 4. 财富格局

包含：
- 财星旺衰
- 身财对比（能否担财）
- 财富层次与来源方式
- 正偏财倾向
- 风险点

### 5. 健康倾向

基于五行偏枯、冲克刑害分析。
**只提倾向，不做诊断**。建议保健方向，不推荐具体药方。

### 6. 事业方向

包含：
- 适合的行业（基于喜用神五行）
- 工作属性（基于十神组合）
- 体制内 / 外倾向
- 创业 / 打工倾向

### 7. 配偶特征

包含：
- 配偶外貌倾向（五行推形象）
- 配偶性格倾向
- 相识方式（基于配偶星位置）
- 相处模式（基于日支）

### 8. 居住与风水

包含：
- 喜用方位
- 喜用颜色
- 居住环境建议
- 办公/卧室环境建议

**不推荐具体风水产品消费，不讲化煞、开光**。

### 9. 学业 / 考试 / 名气

包含：
- 学业能力（印星与食伤）
- 考试运（官印组合）
- 名气潜力（伤官配印、食神吐秀等）
- 适合的学习类型（正统 / 偏门 / 技艺）

### 10. 人生事件节点

基于大运流年列出重大节点：
- 学业高峰期
- 事业上升期
- 财运爆发期
- 感情定型期
- 重大变动期

至少列出 3–5 个明确的人生节点。

## 格式规范

1. **章节标题用 `##` 与 `###`**，层级清晰
2. **表格用于排盘信息**，便于阅读
3. **每个结论要能追溯**到命盘中的具体干支
4. **避免重复声明"仅供参考"**（只在开头说一次即可）
5. **语言风格**：专业中带温度，不装神弄鬼，也不过度娱乐化
6. **篇幅控制**：总字数 3000–6000 字为宜；重点维度详写、次要维度简写
7. **禁止套话**：如"命中注定""前世因果""上天安排"等玄学黑话
8. **禁止绝对化预言**：用"倾向""容易""建议"等词
