# 梦境与八字综合解读

用于用户描述反复梦境时，结合当前流年流月做文化与心理双重解读。

## 输入

- 八字
- 梦境文字描述
- 梦境频率、出现时间、醒后情绪

## 解读流程

1. 提取梦境符号：水、火、路、房子、考试、追赶、坠落、亲人等。
2. 识别情绪：焦虑、遗憾、兴奋、失控、期待。
3. 对照当前流年流月被激活的十神。
4. 输出现实提醒与自我照护建议。

## 常见符号

- 水：情绪、流动、关系变化。
- 火：表达、曝光、压力、行动。
- 房子：安全感、家庭、边界。
- 路：选择、转型、方向。
- 考试：评估、压力、资格。
- 坠落：失控感、节奏过快。

## 合规

梦境解读不是心理治疗。若用户出现自伤、严重抑郁、持续失眠等描述，应建议寻求专业帮助。
