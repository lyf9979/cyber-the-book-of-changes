# 用神判定规则

## 一、日主强弱判断

### 三要素

1. **得令**：日干在月支的状态
   - 旺：月支与日干同五行（如日干甲木生寅卯月）
   - 相：月支五行生日干（如甲木生亥子月）
   - 休：日干生月支（如甲木生巳午月）
   - 囚：日干克月支（如甲木生辰戌丑未月）
   - 死：月支克日干（如甲木生申酉月）

2. **得地**：日干在其他三支（年、日、时）是否有根
   - 本气根最强（如甲木见寅为禄）
   - 中气、余气为次强根、弱根

3. **得生与得助**：
   - 印星（生我者）和比劫（同我者）透干、通根为得助
   - 食伤（我生）、财星（我克）、官杀（克我）多而旺则为身弱

### 强弱等级

- **身旺**：得令 + 得地 + 得助，三者居其二以上
- **身弱**：三者皆失或仅得其一
- **中和**：介于旺弱之间，喜忌较平衡

### 特殊情况

- **从强格**：全盘印比，无财官食伤，从旺气
- **从弱格**（从财、从官杀、从儿）：日主无根，全盘为某一五行，从之
- **化气格**：日干与月干或时干合化成局（甲己合化土、乙庚合化金等），需合化条件具足

## 二、格局判定

### 正格（八格）

以月令本气（或透出之神）定格：
1. **正官格**：月令本气或透出为正官
2. **七杀格**：月令本气或透出为七杀
3. **正财格**：月令本气或透出为正财
4. **偏财格**：月令本气或透出为偏财
5. **正印格**：月令本气或透出为正印
6. **偏印格**：月令本气或透出为偏印
7. **食神格**：月令本气或透出为食神
8. **伤官格**：月令本气或透出为伤官

**禄刃不入格**：月令为比肩（禄）或劫财（刃）时，另以"建禄格""月刃格"论，或改取他柱透神为格。

### 特殊格局

- **专旺格**：曲直（木）、炎上（火）、稼穑（土）、从革（金）、润下（水）
- **从格**：从财、从杀、从儿、从势
- **化气格**：化木、化火、化土、化金、化水
- **双清 / 两神成象格**：命局只见两种五行

## 三、用神选取四法

### 1. 扶抑（身弱扶之，身旺抑之）

- 身弱 → 用印、比扶身
- 身旺 → 用食伤泄秀，或财官克耗

### 2. 调候（根据季节冷暖燥湿）

以日干 + 月令定调候用神：

| 日干 | 冬季（亥子月） | 夏季（巳午月） | 春季（寅卯辰） | 秋季（申酉戌） |
|------|-------------|-------------|-------------|-------------|
| 甲 | 丙 + 癸 | 癸 + 丁 | 丙 + 癸 | 庚 + 丁 |
| 乙 | 丙 | 癸 + 丙 | 丙 | 丙 + 癸 |
| 丙 | 甲 + 壬 | 壬 + 庚 | 壬 | 甲 + 壬 |
| 丁 | 甲 + 庚 | 壬 + 庚 | 庚 + 甲 | 甲 + 庚 |
| 戊 | 丙 + 甲 | 壬 + 丙 | 丙 + 癸 | 丙 + 癸 |
| 己 | 丙 + 甲 | 癸 + 丙 | 丙 + 癸 | 丙 + 癸 |
| 庚 | 丁 + 丙 | 壬 + 戊 | 甲 + 丁 | 丁 + 甲 |
| 辛 | 丙 + 壬 | 己 + 壬 | 己 + 壬 | 壬 + 甲 |
| 壬 | 丙 + 戊 | 癸 + 庚 | 庚 + 戊 | 甲 + 戊 |
| 癸 | 丙 + 辛 | 庚 + 壬 | 辛 | 辛 + 丙 |

> 详见《穷通宝鉴》调候用神表

### 3. 通关（两神相战，取通关之神化解）

- 金木相战，用水通关（金生水，水生木）
- 水火相战，用木通关
- 木土相战，用火通关
- 火金相战，用土通关
- 土水相战，用金通关

### 4. 病药（命局有明显的"病"，取能去病之字为用）

例：日主身弱，印星本应为用，但命中有财星克印——财就是"病"，用比劫克财救印为"药"。

## 四、喜忌神的层级

1. **用神**：最需要的那个字或五行
2. **喜神**：生用神 / 帮用神的字
3. **忌神**：克用神 / 泄用神的字
4. **仇神**：生忌神的字
5. **闲神**：对格局影响不大的字

## 五、实操要点

1. **先看月令**：月令是命盘的"主时节"，定格局的第一依据
2. **再看透干**：天干透出之神作用最显，不透则藏而不发
3. **看干支配合**：天干需地支有根才有力，地支需天干透出才能发挥
4. **看刑冲合会**：冲克会改变五行力量格局，合化可能改变属性

## 六、常见误区

- **不要单纯数五行个数**：五行力量要看位置、得令、透干、有根情况
- **身旺不一定好，身弱不一定坏**：关键看格局是否"成"、用神是否"有力"
- **忌神不一定全是坏事**：忌神若被合化、被克制，反而成为命中"有用之病"
- **运比命重要**：好命坏运不如差命好运，大运流年才是人生轨迹的主导
