# HTML 输出模板（精美展示版）

本模板是所有场景的默认最终输出模板。无论用户触发单人八字、合婚、择日、生育、起名、风水、太岁、梦境或顾问问答，最终都应输出一个**单文件自包含**的 HTML 页面。

只有用户明确要求“Markdown / 纯文本 / 调试文本 / 不要 HTML”时，才改用 `references/output-template.md`。

## 目标

- 用于浏览器展示、分享或打印
- 结构与 `output-template.md` 一致（信息不丢失）
- 视觉风格偏东方审美、克制且可读
- 不以控制台式 Markdown 作为最终报告

## 页面结构（单人八字）

1. 报告头部：标题、副标题、生成时间、免责声明
2. 基本信息卡：公历/农历/性别/出生地/排盘口径
3. 四柱命盘表
4. 五行统计（可视化条形图）
5. 大运时间轴（标注当前大运）
6. 格局与用神摘要卡
7. 十大维度分析（10 张卡片，每项至少 500 字，可折叠展开）
8. 近三年流年（2026-2028，且每年含 12 个月月度表）
9. 总结与执行清单（30 天 / 90 天 / 年度任务 + 复盘机制）
10. 页脚（文化声明 + 非医疗/法律/金融建议）

## 扩展场景页面结构

不同模式应复用同一视觉系统，并按场景添加专属模块。

### 姻缘合婚

1. 双方基本信息卡
2. 双盘四柱并排
3. 合婚总分仪表盘
4. 8 维热力矩阵
5. 协同点 / 摩擦点双栏
6. 未来三年共同节点时间线

### 结婚择日

1. 日期范围说明
2. Top 10 推荐日期卡片
3. 月度热度日历
4. 避开日期清单
5. 现实档期与家庭沟通提醒

### 生育规划

1. 父母命盘摘要
2. 候选出生月份对比表
3. 受孕窗口区间
4. 属相合冲图
5. 子女简化十维卡片

### 手相 / 面相

1. 上传图片缩略图（可 base64 内嵌）
2. 可见特征清单
3. 印证 / 补充 / 矛盾三色标签
4. 八字交叉结论
5. 图像质量提醒

### 起名 / 改名

1. 姓氏与喜用神摘要
2. 候选名卡片
3. 五行补益说明
4. 音形义评分表
5. 谐音与生僻风险提示

### 事业 / 财运择时

1. 当前结构摘要
2. 12–24 个月窗口色块表
3. Top 推荐月份与避险月份
4. 行动清单

### 风水方位

1. 喜用方位摘要
2. 八方位罗盘
3. 家居 / 办公 / 卧室建议卡
4. 颜色材质建议

## 共享组件

| 组件 | 用途 |
|---|---|
| `.dual-chart` | 双盘左右并排，移动端堆叠 |
| `.score-ring` | 0–100 评分圆环，可用 `conic-gradient` |
| `.heat-grid` | 8 维或 10 维热力矩阵 |
| `.month-calendar` | 12 个月择日/择时色块 |
| `.timeline` | 大运、流年、太岁节点 |
| `.compass` | 八方位风水罗盘 |
| `.tag-confirm` | 图像与八字互相印证 |
| `.tag-supplement` | 图像补充八字信息 |
| `.tag-conflict` | 图像与八字存在矛盾 |

## 样式约束

- 不依赖外部 CDN（字体、脚本、样式全部内嵌）
- 响应式：移动端单列，桌面端双列/多列
- 打印友好：`@media print` 去掉阴影，保留主色
- 建议配色（CSS 变量）：
  - 背景：`#f5f1e8`
  - 主文字：`#2b2b2b`
  - 主色：`#8b1e1e`
  - 辅色：`#6b7a52`
  - 边框：`#d8cdb7`
- 五行颜色建议：
  - 金：`#c9a227`
  - 木：`#4f7d39`
  - 水：`#2f5d8a`
  - 火：`#b33a2f`
  - 土：`#8a6a3a`
- 状态色建议：
  - 推荐 / 印证：`#2f7d4f`
  - 中性 / 补充：`#b0852f`
  - 谨慎 / 矛盾：`#9f2f2f`

## 交互与可访问性

- 语义化标签：`header/main/section/footer`
- 图表必须有文字值，不只靠颜色传达
- 对比度满足基础可读性
- 不使用自动播放动画

## 生成要求

- 输出完整 HTML 文档（`<!doctype html>` 开始）
- 仅最小必要 CSS，不使用构建工具
- 不嵌入真实用户隐私信息（示例必须是虚构案例）
- 所有扩展场景底部必须保留文化属性声明、非专业领域声明、重大决策提醒
