# HTML 输出模板（精美展示版）

当用户明确要求“网页展示 / HTML 导出 / 精美版报告”时，除常规文本结果外，可输出一个**单文件自包含**的 HTML 页面。

## 目标

- 用于浏览器展示、分享或打印
- 结构与 `output-template.md` 一致（信息不丢失）
- 视觉风格偏东方审美、克制且可读

## 页面结构

1. 报告头部：标题、副标题、生成时间、免责声明
2. 基本信息卡：公历/农历/性别/出生地/排盘口径
3. 四柱命盘表
4. 五行统计（可视化条形图）
5. 大运时间轴（标注当前大运）
6. 格局与用神摘要卡
7. 十大维度分析（10 张卡片）
8. 近三年流年（3 张年度卡）
9. 综合建议（编号列表）
10. 页脚（文化声明 + 非医疗/法律/金融建议）

## 样式约束

- 不依赖外部 CDN（字体、脚本、样式全部内嵌）
- 响应式：移动端单列，桌面端双列/多列
- 打印友好：`@media print` 去掉阴影，保留主色
- 建议配色（CSS 变量）：
  - 背景：`#f5f1e8`
  - 主文字：`#2b2b2b`
  - 主色：`#8b1e1e`
  - 辅色：`#6b7a52`
  - 边框：`#d8cdb7`
- 五行颜色建议：
  - 金：`#c9a227`
  - 木：`#4f7d39`
  - 水：`#2f5d8a`
  - 火：`#b33a2f`
  - 土：`#8a6a3a`

## 交互与可访问性

- 语义化标签：`header/main/section/footer`
- 图表必须有文字值，不只靠颜色传达
- 对比度满足基础可读性
- 不使用自动播放动画

## 生成要求

- 输出完整 HTML 文档（`<!doctype html>` 开始）
- 仅最小必要 CSS，不使用构建工具
- 不嵌入真实用户隐私信息（示例必须是虚构案例）
