# 流年大运分析方法

## 一、大运与流年的关系

- **大运（10 年一换）**：主导十年的大势与主题
- **流年（1 年一换）**：具体事件应期与临时性影响
- **大运是舞台，流年是演员**：大运定方向，流年应具体事
- 小运、月运影响更细，但本 skill 不展开

## 二、分析顺序

1. 确定当前大运柱
2. 判断大运与命局用忌神的关系
3. 取当前流年及未来 2 年的干支
4. 逐一分析每年：
   - 流年天干与日干的关系
   - 流年地支与命局各支的关系（冲合刑害）
   - 流年与大运的组合作用
   - 对十神的激活情况
   - 对十大维度的具体影响

## 六、"吉凶"判断原则

**吉年的特征**：
- 流年为用神，天干透出
- 流年引动命中的贵人、喜用
- 流年合化成喜用之五行
- 大运流年都为用神

**凶年的特征**：
- 流年为忌神，且大运也忌
- 流年冲克命中的用神
- 流年引动刑冲，动摇根基
- 流年与大运、命局形成"三会""三合"忌神局

**警惕的组合**：
- 岁运并临（流年与大运干支相同）：大事之年，喜则大喜，忌则大凶
- 天克地冲（流年天干克大运/日柱天干，地支相冲）：重大变动
- 反吟伏吟：流年与命中某柱相同（伏吟）或完全相冲（反吟）

## 七、建议输出要求

对每一年：
1. **总评**一句话定调
2. **重点维度**展开分析（不必十维度全写，抓 2–3 个关键的）
3. **月份节点**（可选）：提示这一年哪几个月是关键月
4. **具体行动建议**：宜做什么、避免什么、如何借势

不做绝对断语，用"倾向""容易""建议"等措辞。

---

> 流年分析是命理应用的核心落地点，应具体、可操作、避免空话。
