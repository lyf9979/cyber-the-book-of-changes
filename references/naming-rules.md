# 起名与改名规则

用于宝宝起名、成人改名、艺名/笔名建议。

## 输入

- 八字
- 姓氏
- 性别偏好（可选）
- 风格偏好：古典、清雅、稳重、明亮、现代等
- 避讳字、家族辈分字（可选）

## 分析流程

1. 判断八字喜用神。
2. 确定名字应补益的五行。
3. 从字库中选择对应五行字。
4. 结合音形义、谐音、书写难度筛选。
5. 输出 5–10 个候选名。

## 每个名字说明

- 字义解释
- 五行补益
- 音律与读感
- 与姓氏组合
- 适合场景
- 风险提示（谐音、过冷僻、过网感）

## 合规边界

- 不承诺名字改变命运。
- 不使用贬义、歧义、过度生僻字。
- 尊重用户文化、方言和家族传统。
