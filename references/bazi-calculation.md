# 八字排盘算法参考

## 零、排盘来源与无环境降级

本项目优先追求“可用且不空转”。`scripts/bazi_calculator.py` 基于 `sxtwl`，可作为高精度排盘增强工具，但它不是普通用户使用 skill 的必要条件。

排盘来源按以下顺序选择：

1. **sxtwl local**：本地 Python 与 `sxtwl` 可用时，调用 `scripts/bazi_calculator.py` 生成四柱、藏干、十神、大运与流年。该方式精度最高。
2. **user provided**：用户已经提供完整四柱时，可直接采用用户四柱分析，但必须注明“未重新校验历法”。
3. **LLM approximate**：本地工具不可用且用户未提供四柱时，按本文规则进行近似排盘，并明确标注“近似排盘，仅供文化参考”。

安装熔断规则：

- Agent 最多尝试一次本地排盘工具。
- 如果出现 Python 不存在、`sxtwl` 缺失、C++ 构建环境缺失、安装失败、命令不可执行等情况，立即停止工具尝试。
- 不得反复要求用户安装依赖，不得循环执行 `pip install` 或构建命令。
- 最终报告必须在基本信息或排盘口径中标注排盘来源、置信度与可能误差。

## 二、四柱排法

### 年柱

以**立春**为界，非以正月初一。立春前出生按上一年算。

天干：公元年 - 3，除以 10 取余数对应甲(1)乙(2)...癸(0)。
地支：公元年 - 3，除以 12 取余数对应子(1)丑(2)...亥(0)。

### 月柱

以**节气**换月，非阴历月初：

| 月份 | 节气 | 地支 |
|------|------|------|
| 1 | 立春 | 寅 |
| 2 | 惊蛰 | 卯 |
| 3 | 清明 | 辰 |
| 4 | 立夏 | 巳 |
| 5 | 芒种 | 午 |
| 6 | 小暑 | 未 |
| 7 | 立秋 | 申 |
| 8 | 白露 | 酉 |
| 9 | 寒露 | 戌 |
| 10 | 立冬 | 亥 |
| 11 | 大雪 | 子 |
| 12 | 小寒 | 丑 |

**月干推法**（五虎遁）：根据年干推正月（寅月）干：
- 甲己之年丙作首（甲己年正月为丙寅）
- 乙庚之年戊为头
- 丙辛之岁寻庚起
- 丁壬壬位顺行流
- 戊癸之年何方发，甲寅之上好追求

### 日柱

以 1900年1月1日 干支为基准（甲戌日，序数 11）进行推算，或使用万年历查表。

程序实现建议使用公式法（如 Zeller 公式改编）或直接查万年历。

### 时柱

| 时辰 | 时间 | 地支 |
|------|------|------|
| 子时 | 23:00–01:00 | 子 |
| 丑时 | 01:00–03:00 | 丑 |
| 寅时 | 03:00–05:00 | 寅 |
| 卯时 | 05:00–07:00 | 卯 |
| 辰时 | 07:00–09:00 | 辰 |
| 巳时 | 09:00–11:00 | 巳 |
| 午时 | 11:00–13:00 | 午 |
| 未时 | 13:00–15:00 | 未 |
| 申时 | 15:00–17:00 | 申 |
| 酉时 | 17:00–19:00 | 酉 |
| 戌时 | 19:00–21:00 | 戌 |
| 亥时 | 21:00–23:00 | 亥 |

**子时处理**：
- 23:00–24:00 为晚子时，日柱用当日，时柱干支按"次日日干推时"
- 00:00–01:00 为早子时，日柱归次日，时柱按次日日干推时
- 本 skill 默认使用"00:00 之后归次日"的现代算法，但允许用户指定古法

**时干推法**（五鼠遁）：根据日干推子时干：
- 甲己还加甲（甲己日子时为甲子）
- 乙庚丙作初
- 丙辛从戊起
- 丁壬庚子居
- 戊癸何方发，壬子是真途

## 三、真太阳时校正

若用户提供了出生地经度，可校正时差：
- 北京时间基准：东经 120°
- 时差修正（分钟）= (120° - 当地经度) × 4

例如出生地为东经 114°（武汉附近），修正 = (120 - 114) × 4 = 24 分钟，即实际太阳时 = 北京时间 - 24 分钟。

**建议**：若用户未提供出生地或对精度要求不高，使用北京标准时即可。若出生地远离 120° 经线（如新疆、东北），应主动询问并校正。

## 五、十神关系

以日干为"我"，与其他天干/藏干的关系：

| 关系 | 阳见阳/阴见阴（比和） | 阳见阴/阴见阳（相异） |
|------|---------------------|---------------------|
| 同我（比劫） | 比肩 | 劫财 |
| 我生（食伤） | 食神 | 伤官 |
| 我克（财星） | 偏财 | 正财 |
| 克我（官杀） | 七杀（偏官） | 正官 |
| 生我（印星） | 偏印 | 正印 |

**示例**：日干为戊土（阳土）
- 见戊 = 比肩，见己 = 劫财
- 见庚 = 食神，见辛 = 伤官
- 见壬 = 偏财，见癸 = 正财
- 见甲 = 七杀，见乙 = 正官
- 见丙 = 偏印，见丁 = 正印

## 六、大运起运

### 起运岁数

- **阳年男命 / 阴年女命**：顺排（从月柱下一柱开始）
- **阴年男命 / 阳年女命**：逆排（从月柱上一柱开始）

起运天数计算：
- 顺排：从出生时间到下一个节气的天数
- 逆排：从上一个节气到出生时间的天数
- 每 **3 天 = 1 岁**；1 天 = 4 个月；1 时辰 = 10 天

### 示例

年柱 辛巳（阴年），男命 → 逆排。月柱 己亥，逆推一柱为 戊戌（第一步大运）。

每步大运 10 年。

## 七、神煞（常用速查）

- **天乙贵人**：甲戊庚见丑未，乙己见子申，丙丁见亥酉，壬癸见卯巳，辛见寅午
- **桃花**（咸池）：以年支或日支查：申子辰见酉，寅午戌见卯，巳酉丑见午，亥卯未见子
- **驿马**：申子辰见寅，寅午戌见申，巳酉丑见亥，亥卯未见巳
- **华盖**：申子辰见辰，寅午戌见戌，巳酉丑见丑，亥卯未见未
- **羊刃**：甲见卯，乙见辰，丙戊见午，丁己见未，庚见酉，辛见戌，壬见子，癸见丑
- **文昌**：甲见巳，乙见午，丙戊见申，丁己见酉，庚见亥，辛见子，壬见寅，癸见卯

神煞仅作辅助判断，不作为核心结论依据。
