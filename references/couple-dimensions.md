# 合婚八维分析框架

本文件定义两人姻缘配对报告的固定 8 个维度。

## 1. 性格契合

依据双方日主五行、月令、透干十神判断互动方式。

必须分析：
- 谁更主动，谁更稳定
- 谁偏理性，谁偏感性
- 冲突发生时谁倾向表达，谁倾向回避
- 双方能否形成互补

## 2. 感情稳定

依据双方配偶星、夫妻宫、桃花神煞与大运状态。

必须分析：
- 关系启动快慢
- 长期稳定因素
- 情绪波动来源
- 适合的关系推进节奏

## 3. 家务分工

依据印星、食伤、比劫、财星的结构差异。

必须分析：
- 谁更擅长日常组织
- 谁更适合对外事务
- 谁容易承担过多隐性劳动
- 如何建立公平分工

## 4. 财务配合

依据双方财星、财库、比劫、食伤生财通道。

必须分析：
- 收入方式差异
- 消费观差异
- 风险偏好差异
- 共同账户与个人账户建议

## 5. 健康同步

依据双方五行偏枯与生活节奏。

必须分析：
- 谁更容易过劳
- 谁需要被提醒休息
- 双方作息是否互相影响
- 共同健康习惯建议

## 6. 事业互补

依据官杀、印星、食伤、财星组合。

必须分析：
- 是否适合共同创业
- 是否适合一方主外一方主内
- 对彼此事业的促进点
- 事业压力如何影响关系

## 7. 生育观与子女缘

依据双方时柱、子女星、近年流年。

只做传统命理倾向，不做医学判断。

必须分析：
- 生育观念是否一致
- 对教育方式的差异
- 未来三年是否适合讨论家庭计划

## 8. 矛盾风险

依据天克地冲、刑害破、用神冲突与大运错位。

必须输出：
- 高风险场景
- 早期信号
- 沟通策略
- 避免升级的动作清单
