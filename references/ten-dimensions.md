# 十大维度分析框架

每个维度的分析必须**基于命盘具体干支**，避免套话。

## 一、性格特质

**分析要素**：
- 日干五行基本特质
- 日支藏干（内在性格）
- 月令季节对日干的塑造
- 透干十神的外显特征
- 特殊组合（如杀印相生、伤官配印、财官双美等）

**日干基本特质**：
| 日干 | 基本特质 |
|------|---------|
| 甲木 | 正直、有领导欲、上进心强、略固执 |
| 乙木 | 柔韧、适应力强、依附性、审美好 |
| 丙火 | 热情、光明磊落、喜欢社交、脾气急 |
| 丁火 | 细腻、温和、智慧、略敏感多疑 |
| 戊土 | 稳重、守信、厚道、反应偏慢 |
| 己土 | 柔和、包容、精于算计、有城府 |
| 庚金 | 刚毅、果断、讲义气、易冲动 |
| 辛金 | 精致、敏锐、批判性强、较冷漠 |
| 壬水 | 智慧、豁达、流动多变、贪玩 |
| 癸水 | 聪明、内秀、多愁善感、适应强 |

**十神性格**：
- 正官：正直、守规、有原则
- 七杀：刚烈、有野心、压力大
- 正财：务实、节俭、踏实
- 偏财：大方、人缘好、爱冒险
- 正印：仁慈、学者气、依赖
- 偏印：独特、聪敏、孤僻
- 食神：温和、乐观、有才艺
- 伤官：叛逆、聪明、自负
- 比肩：独立、自我、朋友多
- 劫财：冲动、豪爽、易破财

## 二、感情运势

**分析要素**：
- 配偶星的有无与旺衰（男命以财为妻，女命以官杀为夫）
- 配偶星的位置（年月远、日时近）
- 日支配偶宫的字与喜忌关系
- 桃花、红鸾、天喜等神煞
- 日柱与月柱的关系（是否冲克合害）

**配偶星组合解读**：
- 财星多而杂（男）/ 官杀混杂（女）：感情复杂，易多段
- 配偶星不透藏地支：对象平常，晚婚倾向
- 配偶星透干得力：对象优秀、感情明朗
- 配偶星入墓：感情深藏、易有波折
- 日支逢冲：婚姻宫动荡，易有变动
- 日支逢合：感情缠绵，但也可能"外合"引发桃花

**婚姻时机参考**：
- 配偶星大运流年出现
- 配偶宫被合拱
- 桃花、红鸾流年
- 用神启动之年

## 三、家庭与六亲

**分析要素**：
- 年柱（祖上、远辈、1–16 岁环境）
- 月柱（父母、兄弟、16–32 岁环境）
- 日柱（自身与配偶）
- 时柱（子女、晚年、48 岁后）

**六亲星**：
| 亲属 | 男命 | 女命 |
|------|------|------|
| 父 | 偏财 | 偏财 |
| 母 | 正印 | 正印 |
| 兄弟姐妹 | 比劫 | 比劫 |
| 妻 | 正财（偏财次之） | — |
| 夫 | — | 正官（七杀次之） |
| 子 | 七杀（正官为女） | 食神（伤官为女） |
| 女 | 正官 | 伤官 |

**常见组合解读**：
- 财坏印：父与母不和，或父母缘一方薄
- 比劫重且克财：兄弟姐妹多，或因兄弟破财
- 官杀混杂在月柱：父母关系复杂
- 时柱为用神：子女有出息、晚运好
- 时柱冲克日柱：与子女缘薄或代沟大

## 四、财富格局

**分析要素**：
- 财星旺衰
- 身财对比（能否担财）
- 财星位置（近身 / 远身）
- 食伤生财的通道
- 比劫克财 / 劫财夺财的风险
- 财库（辰戌丑未中藏本五行之财）

**财富层次判断**：

| 组合 | 财富倾向 |
|------|---------|
| 身旺财旺 | 大富命，有担财能力 |
| 身弱财旺 | 富屋贫人，有财难享，需靠印比扶身才能富 |
| 身旺财弱 | 辛苦求财，需食伤生财 |
| 身弱财弱 | 财运一般，求稳 |
| 财库逢冲 | 财富有爆发点，但也易散 |
| 食神生财 | 稳定富足，靠技艺生财 |
| 伤官生财 | 灵活求财，多变动 |
| 七杀得财生 | 偏财、意外财，但有风险 |

**求财方向**：
- 正财：固定工作、稳定收入
- 偏财：生意、投资、兼职
- 食神生财：专业、手艺、创意
- 伤官生财：表达、营销、创新

## 五、健康倾向

**分析要素**：
- 五行偏枯对应脏腑
- 冲克刑害对身体的影响
- 大运流年对健康用神的冲击

**五行对应脏腑**：
| 五行 | 脏 | 腑 | 体系 |
|------|----|----|------|
| 木 | 肝 | 胆 | 神经、筋膜、眼 |
| 火 | 心 | 小肠 | 血液循环、精神 |
| 土 | 脾 | 胃 | 消化、肌肉 |
| 金 | 肺 | 大肠 | 呼吸、皮肤 |
| 水 | 肾 | 膀胱 | 生殖、骨骼、耳 |

**健康风险信号**：
- 某五行过旺或过弱 → 对应脏腑易失衡
- 水火相冲（冬夏之战） → 心肾不交
- 金木相克 → 肝胆呼吸系统
- 命局燥热 → 炎症、血压、上火
- 命局寒湿 → 风湿、循环差、阳虚
- 大运引动忌神 → 旧疾发作

**注意事项**：
- 只提倾向，不诊断
- 不预测大病重病
- 建议用户遵医嘱，命理只作生活方式参考

## 七、配偶特征

**分析要素**：
- 配偶星的五行 → 对方外貌与性格
- 配偶宫（日支）的五行与十神 → 配偶与自己的相处模式
- 配偶星的位置 → 认识方式
- 配偶星距离与强弱 → 感情深度与稳定度

**配偶特征推断**：

**五行推形象**：
- 木 → 身材高挑、偏瘦、文雅
- 火 → 肤色偏红、开朗、较急
- 土 → 敦实、稳重、憨厚
- 金 → 五官立体、冷静、有气质
- 水 → 灵动、聪明、善变

**位置推相识方式**：
- 配偶星在年柱：青梅竹马、远方相识、年长对象
- 配偶星在月柱：经人介绍、同事同学、年纪相近
- 配偶星在日支：一见钟情、自然相识
- 配偶星在时柱：晚婚、小龄对象

## 八、居住与风水

**分析要素**：
- 喜用神五行 → 居住方位
- 忌神方位避让
- 颜色、环境元素建议

**五行方位**：
| 五行 | 方位 | 颜色 | 环境 |
|------|------|------|------|
| 木 | 东 | 青绿 | 植物多、通风、有阳光 |
| 火 | 南 | 红紫橙 | 明亮、干燥、暖色调 |
| 土 | 中央、西南、东北 | 黄棕 | 厚重装饰、陶瓷、稳定 |
| 金 | 西、西北 | 白金银 | 简洁、金属、利落 |
| 水 | 北 | 黑蓝 | 流动元素、安静、冷色调 |

**建议**：
- 若喜火：住所向南、多晒太阳、用暖色、厨房重要
- 若喜水：近水而居、北向房、办公桌放水物
- 若喜木：东向、种植物、木质家具
- 若喜金：西向、装饰金属、空间简洁
- 若喜土：中层、黄色系、陶瓷装饰

**注意**：
- 不推荐具体风水产品消费
- 不讲"化煞""开光"等
- 以环境调节身心为主，不上升到迷信

## 九、学业 / 考试 / 名气

**分析要素**：
- 印星（主学习、文书、学历）
- 官星（主功名、考试、公职）
- 食伤（主才华、表达、口碑）
- 文昌星、天乙贵人等神煞

**学业类型**：
- 正印为用：正统学业，研究学术，科班出身
- 偏印为用：特殊技艺，偏门研究，独特领域
- 食神吐秀：文艺才华，作品出彩
- 伤官配印：才华横溢 + 学历加持，声名显赫
- 官印相生：学而优则仕，考公考编有利
- 伤官见官：叛逆、不走寻常路，考试功名多波折

**考试时机**：
- 印星大运流年 → 学业顺利，考试有利
- 官星大运流年 → 功名机会
- 食伤泄秀 + 印星护 → 才华得到认可
- 伤官见官年 → 考试易失利，需谨慎

**名气判断**：
- 日主通根有力 + 食伤吐秀 + 官印相护 → 易有声誉
- 食神生财 → 在某专业领域有口碑
- 伤官配印 → 才名显著
- 七杀有制 → 权威、影响力

## 十、人生事件节点

**分析要素**：
- 大运十年的主题
- 流年中应事的具体年份
- 冲合刑害引发的事件

**重大节点类型**：

1. **学业节点**：印星出现的大运流年 → 学历提升、考试机会
2. **事业节点**：官杀或食伤透出的运年 → 工作变动、晋升
3. **财富节点**：财星到位 + 身能担财的运年 → 重大财运
4. **感情节点**：配偶星到位、桃花年 → 结婚、恋爱、分手
5. **迁移节点**：驿马星逢冲 → 搬家、出国、调动
6. **冲击节点**：忌神大运叠加流年冲克 → 挫折、健康、破财

**应事原则**：
- 大运管十年大势
- 流年管一年具体事件
- 大运 + 流年组合才应大事
- 天干应天时（机会、决策）、地支应地利（环境、健康）
